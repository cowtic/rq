all

Checks if predicate returns truthy for all elements of the input stream. Iteration is stopped once predicate returns falsey. The predicate is invoked with two arguments: (value, index).

Source:
  • prelude.js, line 147
Example

// With index

// The `matchesProperty` iteratee shorthand.
